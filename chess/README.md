# Echecs (Chess) Game

## Overview

Echecs is a Chess game implemented in C using the raylib library.

## Build Instructions

### Prerequisites

- GCC compiler
- raylib library (included in thirdparty/)
- make

### Building

```bash
make
```

### Running

```bash
./bin/echecs
```

## Project Structure

- `src/` - Source files
- `include/` - Header files
- `assets/` - Game assets (images, fonts)
- `docs/` - Documentation

## Controls

- Mouse click to select and move pieces
- The game follows standard Chess rules

## Documentation

Doxygen documentation can be generated by running:

```bash
doxygen docs/Doxyfile.min
```

The generated HTML documentation will be available in `docs/html/`.
