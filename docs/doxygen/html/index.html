Multi-Mini-Games - Documentation

Introduction

Multi-Mini-Games** est un projet de Licence Informatique (2e annee, 2025-2026) developpe en C avec la bibliotheque raylib. Le projet consiste a creer plusieurs mini-jeux independants, puis a les integrer dans un lobby commun.

Comment compiler

Au niveau racine

make help # Voir toutes les cibles disponibles
make bin # Compiler les bibliotheques + executable lobby
make rebuild-exe # Forcer la recompilation de l'executable
make run-exe # Lancer le lobby
make run-tests # Executer tous les tests
make docs # Generer la documentation

Au niveau d'un mini-jeu

cd suika
make help # Voir les options
make # Compilation par defaut (release)
make MODE=clang-debug # Compilation avec debug/sanitizers
make run-main # Lancer le jeu
make run-tests # Executer les tests

Comment ajouter un nouveau jeu

Etape 1: Creer la structure

Copier le dossier sub-project-example/ ou suivre la structure:

mon-jeu/
├── assets/             # Assets specifiques
├── include/
│   ├── core/           # Logique du jeu
│   ├── ui/             # Interface utilisateur
│   ├── utils/          # Utilitaires
│   └── monJeuAPI.h     # API publique
├── src/                # Fichiers .c
├── tests/              # Tests unitaires
├── Makefile
└── Doxyfile.min

Etape 2: Implementer l'API

Le fichier API doit suivre le pattern defini dans generalAPI.h:
