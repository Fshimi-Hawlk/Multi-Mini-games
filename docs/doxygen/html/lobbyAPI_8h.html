Detailed Description

Public API for the Lobby mini-game.

Author
Fshimi Hawlk
Date
2026-02-07
2026-02-23

This header defines the opaque game handle and the minimal set of functions required to control the game from the lobby.

Note
The internal structure LobbyGame_St is opaque outside this module. The lobby and other modules must never access fields directly. If access to specific state is needed, add dedicated getter/setter functions to this API.
