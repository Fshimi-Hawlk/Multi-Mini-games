This file contains the top-level application loop. It initializes the window and shared resources, runs the lobby, and switches to individual games when triggered (e.g. collision with zone).

Games are loaded on demand via their API (e.g. suikaAPI.h) and run in the same process/window. No separate executables are spawned.
