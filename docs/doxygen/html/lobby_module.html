Module Lobby

Vue d'ensemble

Le Lobby est le module central de Multi-Mini-Games. Il sert de hub de navigation permettant au joueur de se deplacer dans un monde 2D style plateformer et d'acceder aux differents mini-jeux via des zones de collision.

Architecture

Composants principaux

  • LobbyGame_St: Structure principale contenant l'etat complet du lobby
  • Player_st: Entite joueur avec position, physique et skins
  • SubGameManager: Gestionnaire de mini-jeux integres
  • Camera2D: Camera suivant le joueur

Integration des mini-jeux

Zones d'activation

Chaque mini-jeu est associe a une zone rectangulaire (hitbox). Quand le joueur entre en collision avec cette zone, le jeu correspondant est lance.

// Definition d'une zone d'activation

.x = 600,
.y = -150,
.width = 75,
.height = 75

Systeme joueur

Physique

Le joueur utilise un systeme de physique 2D basique:

  • Gravite
  • Deplacement horizontal
  • Saut avec coyote time et jump buffering
  • Collision avec plateformes
