Multi-Mini-Games - Documentation

Introduction

Multi-Mini-Games** est un projet de jeux miniatures independants en C avec raylib. Les jeux sont integres dans un lobby commun - un menu style plateforme 2D.

En Mars 2026, le projet inclut:

  • Un lobby fonctionnel avec mouvement, physique et elements UI
  • Un jeu integre: Solitaire (Klondike) - jeu de cartes classique
  • Infrastructure partagee: logging, types, utilitaires et API
  • Outils de build et documentation

Comment compiler

Compilation racine

make help # Voir les options
make bin # Compiler le lobby
make run-exe # Lancer le lobby

Compiler un jeu

cd solitaire
make help # Voir les options
make # Compiler
make run-main # Lancer le jeu
