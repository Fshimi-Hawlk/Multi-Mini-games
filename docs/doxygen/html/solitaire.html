Overview

Solitaire is a fully-featured Klondike Solitaire card game implementation.

Language

  • Français
  • English

Game Rules

  • Objective: Move all cards to the four foundation piles, sorted by suit from Ace to King
  • Tableau: Build down in alternating colors (red on black, black on red)
  • Stock: Draw cards from the stock pile to the waste pile

Contrôles

  • Left Click: Drag cards between piles
  • Click Stock: Draw a card (or recycle waste pile when empty)
  • N Key: Start a new game

Fonctionnalités

  • Drag and drop multiple cards at once
  • Score tracking (+10 for foundation moves, +5 for tableau moves)
  • Game timer
  • Win detection with congratulations screen
  • Lose detection every 2 minutes

Architecture

Logique du jeu (src/core/game.c)

  • solitaire_init(): Initialize game state
  • solitaire_dealCards(): Shuffle and deal cards
  • solitaire_update(): Game loop update (input, drag & drop)
  • solitaire_checkWin(): Check victory condition
  • solitaire_checkLose(): Check defeat condition

Rendu graphique (src/ui/renderer.c)

  • RenderGame(): Main render function
  • RenderStock(), RenderWaste(), RenderFoundation(), RenderTableauPile()
  • LoadAssets(): Load card textures

Structures de données

  • Card_St: Individual card (suit, rank, face-up/down)
  • Pile_St: Card pile (array of cards, count, position)
  • SolitaireGameState: Complete game state
  • DragState_St: Current drag operation state
