Overview

Solitaire est une implémentation complète du jeu de cartes Klondike.

Language

  • Français
  • English

Game Rules

  • Objectif: Déplacer toutes les cartes vers les 4 fondations (As → Roi), triées par couleur
  • Tableau: Empiler les cartes par ordre décroissant en alternant les couleurs
  • Pioche: Tirer des cartes de la pioche vers la défausse

Contrôles

  • Clic gauche: Déplacer les cartes entre les piles
  • Cliquer sur la pioche: Tirer une carte (ou recycler la défausse quand vide)
  • Touche N: Nouvelle partie

Fonctionnalités

  • Déplacement par glisser-déposer de plusieurs cartes
  • Système de score (+10 fondation, +5 tableau)
  • Minuteur de partie
  • Détection de victoire avec écran de félicitations
  • Détection de défaite toutes les 2 minutes

Architecture

Logique du jeu (src/core/game.c)

  • solitaire_init(): Initialiser l'état du jeu
  • solitaire_dealCards(): Mélanger et distribuer les cartes
  • solitaire_update(): Mise à jour de la boucle de jeu
  • solitaire_checkWin(): Vérifier la condition de victoire
  • solitaire_checkLose(): Vérifier la condition de défaite

Rendu graphique (src/ui/renderer.c)

  • RenderGame(): Fonction principale de rendu
  • RenderStock(), RenderWaste(), RenderFoundation(), RenderTableauPile()
  • LoadAssets(): Charger les textures des cartes

Structures de données

  • Card_St: Carte individuelle (couleur, valeur, face visible/cachée)
  • Pile_St: Pile de cartes (tableau, nombre, position)
  • SolitaireGameState: État complet du jeu
  • DragState_St: État de l'opération de déplacement
