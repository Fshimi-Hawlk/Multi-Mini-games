Detailed Description

@brief Common base structure that **every** mini-game state must embed as its first member.

Layout guarantee:
  struct MyGame {
      BaseGame_St base;

MyGame-specific fields... };

This allows safe downcasting from BaseGame_St* to concrete type only when needed, while the lobby can safely access running, paused, score, and call freeGame.
