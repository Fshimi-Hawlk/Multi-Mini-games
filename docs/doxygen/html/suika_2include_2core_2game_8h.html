Resolves collision between player's circle and a single axis-aligned rectangle.

Performs:

  • closest-point calculation
  • penetration depth computation
  • position correction (push out)
  • velocity nulling along dominant axis
  • ground detection (sets onGround, resets jumps/coyote when landing from above)
Parameters
playerPlayer state (position and velocity are modified)
rectRectangle to collide against
