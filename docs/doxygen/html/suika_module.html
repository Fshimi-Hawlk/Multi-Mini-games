Module Suika (Watermelon Game)

Vue d'ensemble

Suika** (jeu de la pasteque) est un clone du celebre jeu japonais où vous combinez des fruits identiques pour creer des fruits plus gros, jusqu'a obtenir une pasteque. C'est le premier mini-jeu integre au projet Multi-Mini-Games.

Regles du jeu

Regles de base

  • Des fruits tombent du haut de l'ecran
  • Deux fruits identiques qui se touchent fusionnent en un fruit plus gros
  • Le score augmente a chaque fusion
  • La partie se termine si un fruit depasse la ligne rouge

Systeme de physique

Implementation

Le jeu utilise une physique 2D simplifiee pour:

  • La gravite appliquee aux fruits
  • Les collisions entre fruits
  • Les collisions avec les murs du conteneur
  • La detection de fusion

Detection de collision

Systeme de fusion

La fusion se produit quand:

  1. Deux fruits du meme type se touchent
  2. La distance entre leurs centres < somme des rayons
  3. Les deux fruits sont "dropped" (pas en chute libre)
