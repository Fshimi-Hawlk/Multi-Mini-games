Detailed Description

Settings structures for mini-games.

Author
Maxime-CHAUVEAU
Fshimi-Hawlk
Date
2026-02-23
2026-02-23

Provides video and audio settings options that can be passed to game initialization functions. All fields have safe defaults when zero-initialized.

Design principles:

  • Settings is optional (NULL = use defaults)
  • Sub-structures (video, audio) can be NULL individually
  • Macros provide default values for convenience
