\hypertarget{md_translate_autotoc_md11}{}\doxysection{\texorpdfstring{Traduire la documentation}{Traduire la documentation}}\label{md_translate_autotoc_md11}
La traduction utilise \href{https://libretranslate.com}{\texttt{ Libre\+Translate}} en local. Elle se déclenche depuis le bouton {\bfseries{Translate}} de la page d\textquotesingle{}accueil ou depuis le bouton {\bfseries{Translate this documentation}} en haut de chaque page Doxygen.\hypertarget{md_translate_autotoc_md12}{}\doxysubsection{\texorpdfstring{Installation de Libre\+Translate (une seule fois)}{Installation de LibreTranslate (une seule fois)}}\label{md_translate_autotoc_md12}

\begin{DoxyCode}{0}
\DoxyCodeLine{pipx\ install\ libretranslate}

\end{DoxyCode}
\hypertarget{md_translate_autotoc_md13}{}\doxysubsection{\texorpdfstring{Utilisation}{Utilisation}}\label{md_translate_autotoc_md13}
Le serveur PHP et Libre\+Translate doivent tourner en même temps dans deux terminaux séparés.

{\bfseries{Terminal 1 — serveur PHP \+:}}


\begin{DoxyCode}{0}
\DoxyCodeLine{cd\ docs/doxygen}
\DoxyCodeLine{php\ -\/S\ localhost:8080}

\end{DoxyCode}


{\bfseries{Terminal 2 — Libre\+Translate \+:}}


\begin{DoxyCode}{0}
\DoxyCodeLine{libretranslate\ -\/-\/load-\/only\ en,<code\_langue>}

\end{DoxyCode}


Remplace {\ttfamily \texorpdfstring{$<$}{<}code\+\_\+langue\texorpdfstring{$>$}{>}} par le code ISO de la langue souhaitée (ex\+: {\ttfamily fr}, {\ttfamily de}, {\ttfamily es}, {\ttfamily ja}).

Attendre que le terminal 2 affiche \+:


\begin{DoxyCode}{0}
\DoxyCodeLine{Running\ on\ http://0.0.0.0:5000}

\end{DoxyCode}


Puis ouvrir {\ttfamily \href{http://localhost:8080}{\texttt{ http\+://localhost\+:8080}}}, sélectionner la langue et cliquer {\bfseries{Translate}}.\hypertarget{md_translate_autotoc_md14}{}\doxysubsection{\texorpdfstring{Notes}{Notes}}\label{md_translate_autotoc_md14}

\begin{DoxyItemize}
\item Le premier lancement télécharge les modèles (\texorpdfstring{$\sim$}{\string~}300 Mo par langue), ce qui peut prendre quelques minutes.
\item Les téléchargements suivants sont instantanés (modèles mis en cache).
\item Une fois la traduction terminée, recharger la page pour voir le nouveau bouton de langue.
\end{DoxyItemize}
