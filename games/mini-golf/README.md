# Golf 3D (Mini Golf)

## Description

Un jeu de mini-golf en 3D avec une physique réaliste et plusieurs parcours. 

## Auteur

- **Auteur:** Maxime CHAUVEAU
- **Date:** 2026-04-14

## Fonctionnalités

- Physique de balle réaliste en 3D (frottement, rebonds, collisions).
- Système de caméra libre pour bien observer le parcours.
- Différents parcours avec des obstacles variés (pentes, murs, trous).
- Comptage des coups par trou.

## Contrôles

- **Clic Gauche + Glisser** : Ajuster la puissance et l'angle du tir.
- **Relâcher le Clic Gauche** : Frapper la balle.
- **Souris (Mouvement)** : Tourner la caméra autour de la balle.
- **Espace** : Réinitialiser la position de la balle (en cas de blocage).
- **R** : Recommencer le parcours.
- **ESC** : Retour au lobby ou quitter.

## Compilation

Pour compiler et lancer le jeu en mode standalone :
```bash
make
make run-main
```
